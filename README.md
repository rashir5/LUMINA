# Lumina
"Lumina is a full-stack intelligence ecosystem built with React Native and Supabase that proactively manages student productivity through Grok-powered AI and automated academic heuristics."
Core Functionalities
Dynamic Attendance Intelligence: Utilizes custom Bunk-Logic Heuristics to calculate the specific number of lectures required to maintain a 75% safety threshold.

Grok-Powered AI Assistant: A context-aware consultant integrated via the xAI Grok-1 API that analyzes user metadata to deliver personalized academic "survival plans."

Automated Task Extraction: Integrates with the Gmail REST API to identify key academic deliverables and automatically synchronize student task velocity.

Constrained Kanban System: A PostgreSQL-backed project board that enforces Work-In-Progress (WIP) limits to ensure deep focus and prevent cognitive burnout.

Micro-Expense Tracking: Optimized logging for daily campus expenditures with real-time financial reporting and "System Stress" integration.
